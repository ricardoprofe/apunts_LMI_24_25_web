1 Pràctica 2.3. XML

1.1 Exercici 1

Realitzar un fitxer XML amb informació lliure.

El fitxer ha de tindre almenys 3 nivells (comptant amb l’element arrel) i almenys 3 elements principals (productes si estigueren parlant d’una tenda). A més, ha de complir el següent requisit:

  • Tindre pròleg, amb la versió 1.0 de XML i codificació UTF-8.
  • Tindre almenys un valor de cada un dels següents: numèric, string, booleà.
  • Tindre almenys un element buit.
  • Un element CDATA amb informació codificada en HTML o Json.
  • Tindre almenys 2 atributs diferents.
  • El document ha d’estar ben format (es pot comprovar amb un validador de XML) i tindre coherència.

1.2 Exercici 2

Representar el model del document anterior en forma d’arbre. Representar els elements com a rectangles i els atributs com a el·lipses. Es pot utilitzar qualsevol editor de diagrames, com Dia o Draw.io. Exportar la imatge en format png o jpg.
