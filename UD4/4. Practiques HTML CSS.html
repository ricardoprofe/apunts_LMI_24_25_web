1 4. Pràctiques HTML i CSS

En aquesta sèrie d’exercicis faràs una pàgina web estàtica d’una empresa, associació o entitat inventada per tú. Està estructurada en exercicis que es corresponen als continguts de la part teòrica. Cada exercici continua a partir de l’exercici anterior. En acabar un grup d’exercicis, hauràs de fer l’entrega corresponent a eixos exercicis. Aquestes entregues s’avaluaran com a pràctiques.

1.1 Exercici 1. Estructura de directoris

En aquest exercici anem a crear l’estructura bàsica de directoris (carpetes) de les teues pàgines web.

Vés a la teua carpeta Documents i crea un carpeta anomenada public_html. Dins d’aquesta crea una carpeta on anirà la teua pàgina web, anomenada empresa. I dins d’aquesta crea 3 carpetes: img, css i fonts.

1.2 Exercici 2. Plantilla

En aquesta pràctica crearàs la plantilla bàsica de la teua primera pàgina web.

Crea una pàgina sense contingut amb les metadades següents: codificació (UTF-8), autor (author: el teu nom), descripció (description: “nom de l’empresa.”), paraules clau (keywords: “paraules relacionades amb els articles de la teua empresa, separades amb comes”) i viewport.

Comprova que la codificació de la pàgina és UTF-8 (en propietats del document).

Guarda la plantilla amb el nom plantilla_empresa.html dins de la carpeta proves. Torna-la a guardar amb el nom index.html dins de la carpeta empresa.

Edita la pàgina index.html i escriu un text de prova dins de l’etiqueta body de la pàgina. Guarda-la i comprova que es visualitza al navegador.

1.3 Exercici 3. Text

A partir de l’arxiu index.html de la pràctica anterior, heu de crear el contingut d’una pàgina que mostre informació sobre la vostra empresa. La pàgina ha de tindre, com a mínim les següents seccions:

  • Informació de l’empresa: Nom, direcció, sector, etc…
  • Descripció de l’empresa: breu història, característiques, etc…
  • Productes: una breu descripció dels productes que comercialitza, activitats que realitza, etc.

La pàgina ha de tindre els següents elements HTML:

  • Un títol (<title>) amb el nom de l’empresa.
  • Informació sobre la pàgina amb les etiquetes vistes fins ara.
  • Els títols necessaris (atenció a l’ordre de niuament i estructura del document).
  • Paràgrafs, per al text.
  • Elements d’èmfasi: em i strong.
  • Elements semàntics utilitzats correctament, com a mínim <header>, <main> i <footer>. Opcionalment <section>, <article>, etc.
  • Altres elements que considereu necessaris.

1.3.1 Entrega pràctica 1

Comprimeix la carpeta public_html en format zip i entrega-ho a Aules.

1.4 Exercici 4. Propietats de text

Crea un full d’estil extern anomenat general.css per a la la teua pàgina. Guarda’l dins de la carpeta css i enllaça’l a totes les pàgines de la teua web. Defineix les següents propietats d’estils:

  • body:
    • font-family: utilitza una família sans serif.
  • Títols (h1, h2…):
    • font-family: utilitza una família serif o sans serif. Opcionalment, utilitza una font externa, guardada en la carpeta fonts, o una font de Google.
    • font-size: utilitza diversos valors d’em.
    • font-style: normal, italic o oblique. Prova diversos valors segons el nivell de la capçalera.
    • font-weight: Prova diversos valors segons el nivell de la capçalera.
  • p:
    • font-family: utilitza una família serif.
    • font-size: utilitza %.

1.5 Exercici 5. Més propietats de text

Modifica el full d’estil general.css per a afegir les següents propietats d’estils:

  • Títols (h1, h2…):
    • text-align
    • text-decoration
    • text_shadow i text-tranform (opcional)
  • p:
    • text-align
    • line-height i text-indent (opcional)

1.6 Exercici 6: Colors

Modifica el full d’estil general.css per a afegir les propietats de color de text i color de fons als següents :

  • body: text i fons.
  • Capçaleres (h1, h2…): text i fons.
  • p: només text.

Utilitza les 3 formes de codificar les color: noms, valors rgb i hexadecimal. Utilitza Adobe Color o W3Schools color Picker per a triar la combinació de colors.

1.6.1 Entrega pràctica 2

1.7 Exercici 7. Enllaços externs

Crea una nova pàgina anomenada enlaces.html i afegeix 3 enllaços externs, relacionats amb els articles de la teua empresa. Utilitza també les etiquetes <hx>, <p>, etc. necessàries. Crea també un enllaç relatiu a la pàgina index.html.

En la teua pàgina index.html crea un enllaç intern relatiu a la pàgina enlaces.html que acabes de crear.

Els enllaços interns entre les teues pàgines, crea’ls dins de l’etiqueta <nav>.

1.8 Exercici 8. Enllaços locals

Crea una nova pàgina anomenada contacte.html i afegeix la informació de contacte de la teua empresa: direcció postal completa, telèfon (inventat), correu electrònic (enllaç de correu electrònic). Utilitza també les etiquetes <hx>, <p>, etc. necessàries. Crea enllaços relatius a la resta de pàgines del teu lloc web.

En la resta de pàgines del teu lloc web crea enllaços relatius a la pàgina contacte.html que acabes de crear.

Tots aquests enllaços interns han d’estar dins de la etiqueta <nav>.

Crea, en general.css, els estils per als enllaços (colors i text-decoration), prova diversos valors per a a:link, a:visited, a:hover i a:active.

Opcionalment, insereix un mapa de Google Maps amb la direcció de la teua empresa.

1.9 Exercici 9. Llistes

Crea una pàgina anomenada productes.html. En aquesta pàgina crea llistes amb les categories de productes de la teua empresa, amb alguns dels productes més destacats. Utilitza llistes numerades i sense numerar o una combinació d’ambdues.

Opcionalment, utilitza llistes de definició.

Crea estils per a les teus llistes: canvia els marcadors. Opcionalment, usa imatges com a marcadors.

Crea enllaços relatius a la resta de pàgines del teu lloc web, dins de la etiqueta <nav>.

Afegeix els enllaços necessaris per a que la nova pàgina siga accessible des de les barres de navegació de totes les pàgines les del teu lloc web.

1.10 Exercici 10. Imatges

Modifica la pàgina productes.html per a afegir almenys 3 imatges dels productes llistats.

Insereix també una o vàries imatges a la pàgina index.html.

Totes les imatges han de tindre text alternatiu amb la descripció del seu contingut (alt) i han d’estar guardades dins de la carpeta img (empresa/img).

Modifica les imatges amb el Gimp per tal que la seua mida s’ajuste a la teua pàgina. Estableix també el ample i alt amb l’atribut style.

Realitza altres modificacions amb CSS al teu gust: centrar, vores arrodonides, imatges responsives…

1.11 Exercici 11: Imatges de fons

Modifica el full d’estil general.css per a afegir imatges de fons, amb les propietats background.

1.12 Exercici 12: Classes

Crea una classe anomenada destacat i aplica-la a un paràgraf de la teua pàgina index.html. Defineix regles per al tipus de lletra, color, ressaltat, etc. de manera que el paràgraf es diferencie de la resta. Opcionalment, aplica-l a altres paràgrafs de la teua web.

1.13 Exercici 13: span

Dins d’un altre paràgraf de la pàgina index.html, posa una frase dins d’una etiqueta span i aplica-li la classe destacat. En el full d’estil css, crea un estil nou que només s’aplique als span destacats que estiguen dins de paràgrafs, diferent de l’estil de l’exercici anterior.

1.14 Exercici 14: Id’s, estils de llistes

Crea un id anomenat main-nav i aplica-l a l’etiqueta nav de totes les pàgines que tinguen enllaços de navegació interns. Fica els enllaços com a element d’una llista no ordenada i crea els estils css per a que es visualitze en forma de menú horitzontal.

1.15 Exercici 15: Multimèdia

Crea, dins del directori arrel de la teua web, una carpeta anomenada media. Crea una pàgina anomenada multimedia.html dins del directori principal de la teua web. En aquesta pàgina has d’inserir els següents elements:

  • Un arxiu de vídeo (procura que la mida siga reduïda).
  • Un arxiu d’audio.
  • Un vídeo de Youtube

Guarda els dos primer arxius dins de la carpeta media.

Afegeix els enllaços necessaris per a que la nova pàgina siga accessible des de les barres de navegació de totes les pàgines del teu lloc web. Tots aquests enllaços haurien d’estar dins de l’etiqueta <nav>.

1.16 Exercici 16: Creació de taules

Crea una pàgina anomenada horari.html dins de la teua carpeta personal. En aquesta pàgina has de fer una taula amb l’horari de l’empresa.

Utilitza thead i th per a la primera fila de la taula (dies de la setmana) i tbody per a la resta.

Utilitza també els atributs colspan i rowspan per a agrupar algunes hores.

Afegeix els enllaços necessaris per a que la nova pàgina siga accessible des de les barres de navegació de totes les pàgines del teu lloc web. Tots aquests enllaços haurien d’estar dins de l’etiqueta <nav>.

Maqueta amb css la taula de forma que utilitzes les propietats border, border-collapse, text-align, vertical-align i colors de fons i de text.

Aplica també regles per a les vores (border), amplària (width) i altura (height) a la taula i als elements de la taula.

1.17 Exercici 17: Formularis I

Modifica la pàgina contacte.html per a afegir un formulari de contacte amb els següents elements:

  • Nom: input de tipus text.
  • Correu electrònic: input de tipus email.
  • Missatge: textarea.
  • Botó enviar: input de tipus submit.

Utilitza també les etiquetes fieldset i legend. Utilitza el mètode GET.

1.18 Exercici 18: Formularis II

Crea la pàgina registre.html per a fer el registre d’usuaris a la teua web. Crea un formulari (utilitza el mètode GET) amb els següents elements, agrupant-los amb les etiquetes fieldset i legend:

  • Informació personal
    • Nom: input de tipus text.
    • Cognoms: input de tipus text, mida de 40 caracters.
    • Adreça: input de tipus text, mida de 60 caracters.
    • Població: input de tipus text.
    • Correu electrònic: input de tipus email.
    • Telèfon: input de tipus telèfon.
    • Data de naixement: input de tipus data.
    • Sexe: input de tipus radio amb 3 opcions: home, dona i altres.
  • Contrasenya:
    • Contrasenya: input de tipus password.
    • Reescriure la contrasenya: input de tipus password.
  • Informació professional:
    • Nom de l’empresa: input de tipus text.
    • Web de l’empresa: input de tipus url, mida de 60 caracters.
    • Any de fundació: input de tipus number, valor mínim 1500 i màxim 2050.
    • Sectors: input de tipus checkbox, amb almenys 5 sectors (p. e. automoció, mecànica, etc.).
  • Botó enviar: input de tipus submit.

Els camps en negreta són requerits (atribut required).

Afegeix els enllaços necessaris per a que la nova pàgina siga accessible des de les barres de navegació de totes les pàgines del teu lloc web. Tots aquests enllaços haurien d’estar dins de l’etiqueta <nav>.
